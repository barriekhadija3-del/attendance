\## 📋 Project Overview



A fully functional web-based Student Attendance Management System built using PHP, MySQL and Apache (XAMPP). The system allows Admin, Teacher and Student users to manage and view attendance records efficiently.



\---



\## ✅ System Features



\- 🔐 Role-based login (Admin, Teacher, Student)

\- 👥 Student registration and management

\- 👨‍🏫 Teacher management

\- 📚 Course management

\- ✅ Daily attendance marking (Present/Absent/Late)

\- 📊 Attendance reports with percentage

\- 🖨️ Print attendance reports

\- 🎓 Student self-service portal

\## 🚀 How to Run



1\. Install XAMPP and start Apache + MySQL

2\. Clone this repository into `C:\\xampp2\\htdocs\\attendance\\`

3\. Open phpMyAdmin at `http://localhost/phpmyadmin`

4\. Create database called `attendance\_db`

5\. Run the SQL from `database/attendance\_db.sql`

6\. Open `http://localhost/attendance/`
